Entrenador IoT para Lazos de control Idustrial

Proyecto SENA que pretende brindar a los aprendices del área de electrónica y automatización la bosibilidad de desarrollar prácticas de apropiación del conocimiento en lazos de control idustrial, utilizando tecnologías de la Industria 4.0.

Iniciar Sesión

Generalidades de este proyecto

Con este proyecto se podrá monitorear variables de procesos como temperatura, presión y nivel, así como también la visualización del proceso con cámaras IP. De esta manera, se podrá observar el funcionamiento en tiempo real y controlar remotamente cada uno de los lazos de control implementados, produciendo en los aprendices una experiencia más real de lo que sería un proceso de automatización, desde cualquier lugar con conexión a Internet.

Variables de proceso

El lazo de control pretende controlar variables de procesos como temperatura, nivel y presión, a través de un módulo de entrenamiento que permita al Aprendiz SENA, desarrollar prácticas en el mismo.

Internet of Things (IoT)

Para este proyecto se utilizará el módulo IoT ESP32, éste permitirá comunicar el entrenador industrial a Internet, haciendo uso de los servicios que ofrece la plataforma Blynk.

Cámaras IP

Estás cámaras monitorearán a los entrenadores en tiempo real, permitiendo a los aprendices visualizar el proceso físicamente desde Internet.
